# title
A game of rock scissors paper

# description
This file consists of a script to play rock paper scissors. This was built using the while true loop and if else functions.
The script will prompt two users. player1 and player2. It will display a message to the players, stating: pls choose rock, scissors or paper.
The players can only type in rock/paper/scissors because the "valid_choices" was implemented  

# how to use
1. run bash
2. download the file "rock scissors paper"
3. run "rock scissors paper.py" in bash

# example
player1 pls choose rock, scissors and paper paper
player2 pls choose rock, scissors and paper scissors
player1 wins!
